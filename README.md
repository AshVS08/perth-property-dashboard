# Perth Property Market Dashboard

An end-to-end data analytics project analysing Perth suburb property values using Python and Power BI.

## Project Overview

This project collects, processes, and visualises Perth property market data to answer the question:

> "Which Perth suburbs offer the best value based on price, location, and amenity factors?"

## Tech Stack

- Python 3.13
- Pandas
- Power BI
- ABS API
- Kaggle Perth House Prices dataset

## Scripts

| Script | Purpose |

| `get_abs_data.py` | Fetch Perth LGA population from ABS API (2018-2023) |
| `process_property_data.py` | Clean and process 33,549 property transactions |
